# 📊 SQL Data Analytics Project

Welcome to my **SQL Data Analytics Project** 🚀\
This project is a collection of SQL scripts that I created to explore,
analyze, and extract insights from relational databases.

It demonstrates my hands-on skills in **SQL, Data Analysis, and Business
Intelligence**, and showcases how I approach real-world data problems.

------------------------------------------------------------------------

## 📖 Project Overview

In this project, I focused on writing optimized SQL queries to perform
different types of analysis, including:

-   Database exploration\
-   Data cleaning and transformation\
-   Metrics and KPI calculations\
-   Time-based trend analysis\
-   Cumulative analytics\
-   Customer and product segmentation

Each script is designed to solve a specific analytical problem and
follows best practices for readability and performance.
